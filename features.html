Product tagging

A placed tag exactly on the loction of the product on the screen, clickable for addition information. Follow a product throughout the video and make it available for sales.​

People tagging

A placed tag exactly on the face of the person on the screen, clickable for addition information such as the biography and with links to Instagram, Facebook and/or Linkedin.

Picture in picture

Additional videos can be presented to and played by customers. The additional videos overlays the paused original video which continues when playback is cancelled.

Clickable banners

Graphical banners with a transparent layer can both be pushed to the screen while playing the video or pulled when video is paused. Placement anywhere on the screen, all iad versions supported.

Response forms

Generate leads by having viewers leave contact details – directly triggered in the video and connected to your environment for easy data access.

Call-to-action

A two-stage form that can be pushed to the front to inform the user an action can be taken in that time segment. Click the button and a message appears such as an embedded video, voucher or contact form.

Save as a favorite: person

Viewers can save the person to their favorite list for later access allowing them to continue to watch the content and come back later. The favorite list data is also available as API for direct integration in the website.

Save as a favorite: product

Viewers have the option to save the product to their favorite list for later access allowing them to continue to watch the content and come back later. The favorite list data is also available as API for direct integration in the website.

Save as a favorite: scene

With the add to watch list button the viewers are able to remember a moment in time of the programme, it registers the exact time and stores the frame of the video for reference. This data is also available with the API.

Main menu

Default placed in the top of the screen but can be positioned anywhere and as well vertical. The main menu can be switched off, as such is each individual button in the menu.​

Main menu: product overview

The quick access product button let the user access instantly all the products that are tagged in the programme. Making it easy to find something back​.

Main menu: people overview

The quick access custom product button let the user access instantly see all the people that are tagged in the program in an easy scroll overview. Each person can be clicked for additional details and can be added to the favourite list.​

Main menu: custom product overview​

The quick access custom product button let the user access instantly all the products placed on a separate list. These products have the option to instantly connect to an API and fill the shopping basket of a website with all the products in that category.

Main menu: live poll​​

Serve up live data through voting or audience participation., Let them instantly engage with the content during the broadcast. The data is available during and after the broadcast.

API favorites​​

With the secure API you are able to get the favorite listing of the viewer connected to your user-id to incorporate in your website. With this option you are able to inform the user with upcoming events connected to the items on the favorite list.​
